# Release binaries


Until GitHub Actions CI/CD is set up, here are pre-built binaries for all OSes, built with the Dockerfile.  

Refresh this folder with `make docker-build-all` from the repo root (builds **sfu** and **sfs** in Docker, then copies `bin/` → `release-bins/`). Commit and push when ready.

You are of course welcomed to build them yourself, full instructions are included in the main README.
